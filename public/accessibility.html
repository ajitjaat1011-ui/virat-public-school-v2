Our commitment

Virat Public School is committed to ensuring digital accessibility for people of all abilities. We aim to conform to Web Content Accessibility Guidelines (WCAG) 2.1 Level AA across this website.

Measures we have taken

  • Semantic HTML5 structure for screen-reader compatibility
  • Visible focus indicators on all interactive elements
  • Keyboard navigation support throughout the site
  • Sufficient color contrast for text and interactive elements
  • Alternative text for all meaningful images
  • Captions and labels for form fields
  • Skip-to-content link for keyboard users
  • Resizable text without layout breakage
  • Reduced-motion support for users who prefer minimal animation

Known limitations

While we strive for full WCAG 2.1 AA conformance, the following known limitations exist:

  • No dark mode in Phase 1. A dark colour scheme is planned for a future release. The site is built on a light theme consistent with the school's brand.
  • Gallery photographs use decorative gradients as placeholders for actual school photos, which will be uploaded upon commissioning of brand photography.
  • Punjabi / Hindi support (Phase 1.5) is not yet available. The site is currently English-only.

Feedback

If you encounter an accessibility barrier on this website, or if you need information in an alternative format, please let us know. We will respond within 7 working days and work to address the issue as quickly as possible.

Assessment

This website is tested using a combination of automated tools (Lighthouse, axe) and manual testing, including keyboard-only navigation and screen reader testing with NVDA and VoiceOver. We aim to re-audit the site quarterly and after any major design changes.
